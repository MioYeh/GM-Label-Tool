# GM-Label-Tool
The tool for label General Movements(GM) videos

## Need to install
```
pip install pyqt5
pip install opencv-python
```

## Git Clone in your computers
```
git clone https://github.com/MioYeh/GM-Label-Tool.git
```

## Run
```
cd GM-Label-Tool
python main.py
```
